# Accordion

### [Demo](https://accordion-challenge.vercel.app/)

## Development

### Install

```shell
yarn install
```

### Start develop server

```shell
yarn dev
```

### Build

```shell
yarn build
```

### Preview

```shell
yarn preview
```

## Environment variables

| Key                  | Default  | Options              | Description                          |
| -------------------- | -------- | -------------------- | ------------------------------------ |
| VITE_POST_REPOSITORY | `memory` | `memory`, `fake_api` | Repository to use in post repository |
